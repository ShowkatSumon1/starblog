== Description ==

Starblog is a blank canvas for your ideas and it makes the block editor your best brush.

With new block patterns, which allow you to create a beautiful layout in a matter of seconds, this theme’s soft colors and eye-catching — yet timeless — design will let your work shine.

Take it for a spin! See how Starblog elevates your portfolio, business website, or personal blog.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in starblog in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Create Home and Blog page. Keep it blank.
5. Go settings > reading > select Home for home page and Blog for blog page.
6. Navigate to Appearance > Customize in your admin panel and customize to your taste.

== Privacy ==
Starblog uses LocalStorage to save the setting when Dark Mode support is turned on or off.
LocalStorage is necessary for the setting to work and is only used when a user clicks on the Dark Mode button.
No data is saved in the database or transferred.
